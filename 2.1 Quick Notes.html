Quick Notes

怎么启用快速笔记？

让你的灵感随时记录下来。

在设置中，设定快速笔记存放的文件夹后即会启用。
 (左键)点击托盘按钮 (Mac版为顶部菜单栏的按钮)后，会有个快速地笔记输入区域。除了普通的笔记之外，还有专门一栏为Code，可以直接选定某类程序语言，进行代码片段的保存。

Mac 版还有一栏为 App，在不同的 App 切换的时候，会自动对应一个当前 App 名字的文件夹以存储。

App 栏中支持下载网页？

是的，Mac 版中的 App 一栏中，可以对 Google Chrome 与 Safari 两个浏览器进行网页转为 Markdown 的抓取。除了将当前 URL 对应的 HTML 内容转为 Markdown 保存之外，还会自动下载内部的图片到本地。

注意: 网页转为 Markdown 的工作机制无法保证完美的转义，多数时候、以内容为主的页面，会有很好的表现，特别本身也是用 Markdown 完成书写的网页内容。

如何关闭快速笔记？

在设置中，清空快速笔记存放的目录即可。
