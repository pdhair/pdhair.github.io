Pandoc

注意!!!

Pandoc是文档转换的瑞士军刀，但这是非常高阶的应用，包括相关联的 LaTeX 更是非常非常高阶的应用，而MarkEditor 仅仅是做了常用命令的 GUI (视觉交互)的集成。
 这能提升日常使用中的效率，但也意味着，你仍然需要自己安装 Pandoc 等相应的程序，在文档转化的过程中，如果有异常、错误需要自己解决。
 如果有更多的自定义要求，包括导出 PDF 时就需要修改、创建自己需要的 LaTeX 模板。
 甚至有可能因为运行 Pandoc 的过程中 (比如你自己定义的 LaTeX 模板错误)，导致整个 MarkEditor持续无法响应 (导出过程中会短时无法响应)，则需要强制停止程序进程并重启。

在使用 Pandoc 导出文档的时候，窗口内的一些设置项，比如字体、PageStyle 都是 LaTeX 模板内的相应概念，不另外具体说明；并且生效的前提是当前使用的(LaTeX)模板源码内有对应的设置项可供替换，否则也是不能生效的。

Pandoc 安装一般说明

参考 Pandoc 官网的安装说明 http://pandoc.org/installing.html.
 一般情况，会有这些步骤:

  1. 下载安装pandoc
  2. 安装LaTeX
  3. tlmgr install titling && tlmgr install lastpage

一般情况下，安装 LaTex 之后，需要让它的运行目录在系统内可以直接调用，比如 Mac 下的地址一般为/Library/TeX/Distributions/.DefaultTeX/Contents/Programs/texbin。
